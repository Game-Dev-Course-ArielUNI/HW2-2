# HW2-2
זה משחק שמראה דג ששוחה בים כאשר בצד ימין למעלה אפשר לראות את המיקום שלו במפה בmini map.
לגבי השאלה:

# בטאב ה"Simulator" של עורך יוניטי, אפשר ללחוץ על הכפתור Rotate כדי לסובב את הסימולטור, וכך לדמות מצב שהשחקן מסובב את הטלפון שלו ממצב אופקי לאנכי ולהיפך. שימו לב, שכאשר אתם מסובבים את הסימולטור למצב אנכי, כל העצמים בסצנה נראים גדולים יותר,  הסבירו מדוע (רמז: זה קשור לתכונה Size של המצלמה).


כאשר את היחס בין רוחב לגובה משתנה (Aspect Ratio),
הערך הזה משתנה:

aspect = Screen.width / Screen.height


לדוגמה:

במצב אופקי (Landscape) – היחס הוא בערך 16:9 → aspect ≈ 1.77

במצב אנכי (Portrait) – היחס הוא בערך 9:16 → aspect ≈ 0.56

כלומר, כשעוברים לאנכי, הערך קטן מאוד.

התכונה Size של המצלמה קובע כמה יחידות נכנסות לגובה המסך , כאשר המסך נהיה צר יותר,הרוחב קטן ואנחנו רואים את העצמים גדולים יותר.
